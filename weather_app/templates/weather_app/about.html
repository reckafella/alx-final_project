About Weather App

This is a simple weather app that uses data from WeatherAPI.com to display current weather and 7-day forecast for any given city / location.

It is built using Django, HTML, CSS and WeatherAPI.

It is part of the final Project for the Software Engineering Program at ALX Africa.
